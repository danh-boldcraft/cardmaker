# Memberstack's Admin REST API

The Memberstack Admin REST API provides powerful server-side capabilities for member management, token verification, and administrative tasks. This secure API allows you to integrate Memberstack's functionality into any server-side environment, regardless of programming language.

## Key Features

### Complete Member Management

Create, read, update, and delete members through a secure server-side API. Manage plans, custom fields, and member data programmatically.

### Token Verification

Securely validate member authentication tokens server-side to protect sensitive resources and verify member identity.

### Language Agnostic

Use with any programming language that can make HTTP requests. Perfect for PHP, Ruby, Python, Go, Java, or any other server-side technology.
